## JAR 이란?

- 자바 프로젝트 압축 파일
- ZIP 파일 압축 알고리즘을 기반으로 만들어짐
- 자바로 개발한 여러 클래스 파일들 또는 패키지 파일이 있을 때, 이를 하나로 묶으면 참조하기 편하다.
