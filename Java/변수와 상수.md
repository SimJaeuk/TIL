# 변수와 상수


## 변수란?

- 단 하나의 값을 저장할 수 있는 메모리 공간
- 실수형(float, long), 정수형(byte, short, int, long), 문자형(char) 등이 있음
- 나중에 값을 변경하게 되면 가장 마지막에 수정한 값이 변수의 값이 됨

## 상수란?

- 수정이 불가능한 값
- 선언 시 초기화 하지 않았다면 단 한번 정할 수 있음
- 변수 앞에 final을 붙여서 선언
