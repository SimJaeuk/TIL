# 일반변수와 참조변수
### `일반변수는 값을 가지고 있고 참조변수는 주소를 가지고 있다.`

## 일반변수

- int, double, char 등(자료형)으로 선언된 변수
- 컴파일시 미리 크기를 알 수 있다.


## 참조변수

- Bean등 임의로 만든 Class자료형
- 컴파일시 크기를 알 수 없다.
