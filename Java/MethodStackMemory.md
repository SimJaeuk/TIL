## 스택

- LIFO 구조, 마지막에 저장된 것을 제일 먼저 꺼냄
- 입력과 출력이 한 방향으로 제한
- 데이터를 넣는 작업을 푸쉬(push)라 하고, 꺼내는 작업을 팝(pop)이라고 함


## 큐

- FIFO 구조, 제일 먼저 저장한 것을 제일 먼저 꺼냄
- 데이터를 넣는 작업을 오퍼(offer)라 하고, 꺼내는 작업을 폴(poll)이라고 함
- 데이터를 꺼내는 쪽을 프론트(front), 넣는 쪽을 리어(rear)라고 함
